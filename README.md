# DataGenius.in

**India's premier data recovery knowledge hub, software suite, and subscription platform.**

> "Your Data. Recovered. Protected. Mastered."

---

## About

DataGenius is a full-stack data recovery media & tools ecosystem — 3 software products, a paid newsletter, free web tools, a blog, a recovery lab directory, and a course academy, all under one brand.

## Tech Stack

- **Framework:** Next.js 16 (App Router) + TypeScript
- **Styling:** Tailwind CSS (dark mode default)
- **Hosting:** Vercel (recommended)

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
